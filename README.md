# uniauth

#### 介绍
{**以下是Su360统一认证管理平台说明**
uniauth是一个基于OAuth2的认证服务以及资源授权服务的管理平台，主要用于系统权限认证以及资源授权访问。}

#### 软件架构
软件架构说明
主要使用的组件为：springboot、spring security、mybatis、OAuth2、redis
使用的数据库为：mysql
数据库连接池为：druid
使用的缓存为：redis
